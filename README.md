## 🚀 Quick Start

```bash
# Backend
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload

# Frontend
cd frontend
npm install
cp .env.example .env
npm run dev
```

Visit: `http://localhost:5173`
